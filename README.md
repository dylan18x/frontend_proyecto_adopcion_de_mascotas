Instalación
Clonar el repositorio:
git clone https://github.com/dylan18x/frontend_proyecto_adopcion_de_mascotas.git
cd frontend_proyecto_adopcion_de_mascotas
code . 

Instalar dependencias:
npm install
Configurar variables de entorno: Crea un archivo .env en la raíz del proyecto.

Variables de Entorno
Crea un archivo llamado .env en la raíz del proyecto y añade esto:

VITE_API_URL=https://fernandez-adopcion-mascotas-api.desarrollo-software.xyz

Ingresar a la URL en la que se realizo el despliegue
https://huellitas-felices.desarrollo-software.xyz/

Credenciales de Prueba
Para acceder al panel admin, utiliza las siguientes credenciales:

usuario: admin 
contraseña: admin123
